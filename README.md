# M-CMP mc-workflow-managerThis repository provides a Workflow Manager.A sub-system of [M-CMP platform](https://github.com/m-cmp/docs/tree/main) to deploy and manage Multi-Cloud Infrastructures.## OverviewM-CMP의 mc-workflow-manager 서브시스템이 제공하는 기능은 다음과 같다.- 워크플로우 생성 기능- 워크플로우를 통한 멀티 클라우드 인프라 생성 및 애플리케이션 배포 기능## 목차1.
